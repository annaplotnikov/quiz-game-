# ❓ Quiz Game — 3 languages

A multiple‑choice trivia game implemented in **Python**, **JavaScript (browser)**, and **C++ (console)**.

## 🎮 Play / Run

| Language | File | Run command |
|:---|:---|:---|
| Python | `python/quiz.py` | `python quiz.py` |
| JavaScript | `javascript/index.html` | Open in any browser |
| C++ | `cpp/quiz.cpp` | `g++ quiz.cpp -o quiz && ./quiz` |

## 📝 How to play

- Answer 5 multiple‑choice questions.
- Each correct answer = 1 point.
- Final score is displayed at the end.
